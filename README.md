# Sistema de Gestão Empresarial

Projeto desenvolvido para gestão operacional de obras.

## Funcionalidades

- Cadastro de empresas
- Cadastro de colaboradores
- Controle de obras
- Gestão de orçamentos
- Integração com aplicativo móvel

## Tecnologias

- Oracle Apex
- SQL Server
- JavaScript
- HTML
- CSS

## Desafio
A empresa precisava centralizar a gestão de obras,
colaboradores, orçamentos e equipes de campo.

## Solução
Foi desenvolvido um sistema web para gestão operacional,
integrado a aplicativo móvel para consulta das informações.

## Resultado
Centralização das informações operacionais e melhoria no fluxo de comunicação entre escritório e equipes de campo.

## Minha atuação

- Desenvolvimento de funcionalidades
- Modelagem de banco de dados
- Implementação de telas
- Integração entre sistemas
